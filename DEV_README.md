Drizzle-ORM is used to establish tables and relationships

Drizzle-kit is used for development to view tables and relationships
and used to initialize the config.

npm install drizzle-orm
npm install drizzle-kit
npx drizzle-kit push:pg
npx drizzle-kit studio
